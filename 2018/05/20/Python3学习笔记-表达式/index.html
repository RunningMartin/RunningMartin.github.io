    解包

    • 交换变量，3个以内，优化为ROT指令，直接交换栈帧数据。

    • 大于3个，则建立元组，进行解包来实现。

星号收集

  • 一个*代表列表，两个*代表字典。收集的时候优先满足非*的参数。

  • 使用时，只能有一个星号，不能出现多个星号，因为不能确定范围。

  • 星号不能单独出现。

星号展开

  • 可以用来展开可迭代对象。

  • 对于字典而言，一个*展开主键，两个*展开键值。

作用域

  • 所有的操作默认为本地名字空间。

  • 声明调用全局名字空间的变量:global x。

  • 声明调用非本地名字空间的变量:nonlocal x。

  • nonlocal自内向外依次检索嵌套函数。但是不包括全局名字空间。由于其是在编译期绑定的，因此目标变量必须提前存在。

  • 可以采用global创建全局变量。

  • 不能采用nonlocal创建非本地名字空间变量。

运算符

  • @functools.total_ordering:基于__eq__和__lt__，补全__lt__(),__le__(), __gt__()和 __ge__()。

    逻辑运算

  • and:返回最后或导致短路的操作数。

  • or:返回第一真值或最后的操作数。

  • 编译分析and

  • 编译分析or

    条件表达式

    • 表达式:T if x else f,如果x为真则返回T，否则返回F。

    控制语句

    • 选择语句：需要通过各种方法来减少选择语句和分支，减少缩进层次。

      • 将过长的选择语句或分支变为函数。

      • 借助代码分析来去除死代码。

        • 循环
          • for：对对象进行迭代。
          • while:执行逻辑循环。在while中可以采用else，在while正常结束后会执行(不是采用break、return进行中断)。

    推导式性能分析

    • 编译器将推导式编译为函数，然后通过LIST_APPEND指令直接添加数据。
