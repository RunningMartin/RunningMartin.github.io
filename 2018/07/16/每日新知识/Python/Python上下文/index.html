上下文管理协议是对异常结构的一种包装。其最大的作用为将用户的逻辑与相关的上下文处理相分离。

原理

  • 表达式：表达式创建的对象必须包含__enter__和__exit__协议方法。这样上下文的开启和结束是与用户代码无关的。

    • __enter__：在执行用户代码之前的操作。
    • __exit__：执行完毕后的清理操作。
