Linux--文件与目录管理

Posted on 2018-07-22 Edited on 2019-10-26

文章摘要

目录操作

  • 特殊目录:
    • .：当前目录
    • ..：上一级目录
    • -：前一个打开的工作目录
    • ~：当前用户主目录
    • ~account：account用户的主目录

切换工作目录

  • cd

显示当前工作目录

  • pwd

创建目录

  • mkdir

删除目录

  • rmdir

执行路径

  • 执行路径用来搜索执行的命令的位置，如果命令不在这些路径中，则不能直接通过名字执行，必须采用完整的路径来执行。.不要放在执行路径中，这是因为如果有人在共享文件夹中做了一个恶意程序，程序的名字为常用的命令名字，如ls，这会导致在共享文件中执行ls时会调用恶意程序。

文件与目录管理

  • ls命令

  • cp命令

  • rm命令

  • mv命令

  • basename：取完整路径的最后一段。

  • dirname：取完整路径的文件夹。

文件查看

  • cat

  • tac：倒序

  • nl：添加行号定义

  • more：more命令可以一页一页的读，但是不能往上翻页。

  • less：和more类型，但是可以往上翻页。

  • head：取前几行。

  • tail：取最后几行。

  • od：打开二进制文件。

文件信息修改

  • mtime(modification time):内容修改时间
  • ctime(status time):状态修改时间，如权限修改，。
  • atime(access time):文件内容被读取时会修改。
  • touch:修改文件时间。

默认权限

  • umask
  • 文件默认属性666，文件夹默认属性777

隐藏属性

  • chattr
  • lsattr

文件查找

  • file:查看文件类型。

  • which:指令搜索。

  • whereis:只在特定的目录(可以通过whereis -l 查看)中查找，一般用于查找命令或系统自带的文件夹。

  • locate:通过建立在/var/lib/mlocate/中的数据库查询。

  • find:
