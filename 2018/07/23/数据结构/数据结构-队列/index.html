队列

  • 概念:队列为只允许在一端进行插入一端进行删除的线性表。是一种先进先出的线性表。

  • 操作

    清空 ClearQueue clear
    判空 IsEmpty is_empty
    取队头 GetTop get_top
    删除 Delete delete
    插入 Insert insert
    长度 Length length

循环队列

  • 由于直接采用单向队列的话，入队的时间复杂度为$$O(1)$$，但是每次出队都是在下标为0出处理，所以每次出队后，都要往前移动，因此出队的时间复杂度为$$O(n)$$，为了解决这个问题采用一些方法来使队列首尾相接(循环队列)。其插入和删除操作的时间复杂度为$$O(1)$$。

  • 方法:假设队头指针front，队尾指针rear。

    • 队空:$$front=rear$$
    • 对满:$$(rear+1)%size=front$$
    • 队的长度:$$(rear-front+size)%size$$

链式队列

  • 循环队列可能面临数组溢出问题，因为我们可以采用带头结点(便于插入和判断队空)的链式队列进行存储。

  • 其插入和删除的操作是$$O(1)$$
