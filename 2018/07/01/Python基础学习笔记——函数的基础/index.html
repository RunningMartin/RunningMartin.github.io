Python基础学习笔记——函数的基础

Posted on 2018-07-01 Edited on 2019-10-26 In Python3 学习笔记

记录关于Python函数基础的学习笔记。

定义

  • 函数的作用：函数是代码复用和模块化的基本单位，用于执行单一逻辑(单一职责，降低代码复杂度)。

  • 函数的设计要求：

    • 单一职责(即只做一件事)。
    • 降低对外界的依赖关系(便于测试和维护)。

  • 创建

    • 概念：
      • 名字：在名字空间中与函数对象相关联，因此，如果函数定义的时候有装饰器，其装饰器的返回值是与名字相关联的。也因为这个特性，函数无法实现重载。
      • 函数对象：为上下文提供调用实例；管理所需要的状态数据。函数对象作为外部示例的存在，负责管理运行期状态。
      • 代码对象：持有字节码和指令元数据。代码对象只关注执行。
    • 总结：def为运行期指令，以代码对象为参数，创建函数实例，并在当前上下文中与指定的名字相关联。因此def可以通过单一代码对象创建多个函数实例。

参数

  • 种类：

    • 位置参数
    • 键值参数

  • 传参：在python中，参数分为形参和实参，形参为局部变量，只能在函数中使用，当函数调用结束后及销毁。采用复制的形式将实参的值传递给形参。因此当传递的值为类似list这一类的引用时，函数中的改变会影响外部。
