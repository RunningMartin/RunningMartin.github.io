数据结构--线性表

Posted on 2018-07-21 Edited on 2019-10-26 In 数据结构

记录在学习数据结构的线性表的笔记，采用C和Python实现。

概念

  • 线性表：零个或多个数据元素的有限序列，这也就意味着除第一个元素以外，每个元素有且仅有一个直接前驱，除最后一个元素以外，每个元素有且仅有一个直接后继。数据元素的个数n为线性表的长度（缺图）

  • 基本操作：

    初始化 InitList __init__
    判空 IsEmpty is_empty
    清空 ClearList clear
    插入 InsertElem insert
    删除 DeleteElem delete
    访问 GetElem get
    定位 Index index
    长度 Length length

顺序存储结构

  • 顺序存储通过一段地址连续的存储单元依次存放线性表的数据元素。实质为通过数组进行存储。

  • 地址计算：$$A_i = A_1 + (i-1)*d(d为每个元素的大小)$$

  • 特性

    • 存取时间复杂度O(1)，因为能通过公式直接计算出元素的位置。
    • 插入删除时间复杂度为O(n)，因为需要移动元素。
    • 长度变化很大时容易造成空间浪费。没有真正的动态数组，动态数组是通过固定长度的数组实现的。

链式存储结构

  • 链式存储是为了解决多删除或插入的情况下，顺序存储的会消耗很多时间，空间容量难以确定的问题。其思路是通过在结点中添加一个指针域，存储下一个结点的地址，这样就不需要连续的内存空间。其插入元素的时间复杂度为$$O(1)$$，读取的时间复杂度为$$O(n)$$。

  • 头指针：指向整个链表的第一个结点。

  • 头结点：如果没有头结点的话，第一个元素的插入和删除会比较特殊化，为了简便操作，可以通过添加头结点，浪费一个结点的空间来换取操作的统一。

    单链表

  • 单链表：单向遍历的链表。每个结点只有一个指针域，指向下一个结点。

静态链表

  • 静态链表是通过数组来实现的链表，其采用的是通过在每个元素中设置一个游标位，游标位存储下一个结点在数组中的位置。数组的第一个元素作为空结点链表的头结点，其游标指向空结点链表的第一个元素的下标，数组的最后一个元素作为单链表的头结点，其游标为单链表第一个结点的下标。
