Python基础学习笔记——函数的闭包

Posted on 2018-06-25 Edited on 2019-10-26 In Python3 学习笔记

Python3基础之函数的闭包。

什么是闭包

闭包是函数在离开其生成环境后，仍然可以记住并持续引用外部变量。闭包有很多不方便的地方，使用前请多多考虑。

特性

  • 自由变量
  • 自引用形成闭包
  • 延迟绑定：闭包只是绑定引用的变量，并不会立即计算引用的内容，只有执行时才会访问所引用的对象。

分析

  • 字节码分析
  • 延迟绑定分析

优缺点

  • 优点
    • 封装了一些特性，可实现隐式上下文状态，减少了参数。
    • 可替代全局变量。
    • 将执行环境与调用接口相分离。
  • 缺点
    • 对自由变量的隐式依赖。
    • 提升了代码复杂度。
    • 生命周期的延长，提高内存占用。
